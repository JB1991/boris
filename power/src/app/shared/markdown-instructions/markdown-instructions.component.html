Für die Formatierung und Gliederung von Texten wird "Markdown" verwendet. Eine Einführung finden Sie unter: Wikipedia

Es ist eine Auszeichnungssprache und ist im Internet im technischen Umfeld verbreitet. Lesen Sie weitere Details unter der folgenden Seite: https://daringfireball.net/projects/markdown/syntax
